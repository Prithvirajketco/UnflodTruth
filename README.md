# UnflodTruth

Welcome to **UnflodTruth**, a powerful tool designed to help uncover and verify factual information in real-time. 

## Acknowledgements

We would like to acknowledge that this project was built upon and enhanced using the open-source **[InTruth](https://github.com/rpanigrahi222/intruth-factcheck)** project. Their work on real-time fact-checking and live claim evaluation provided an excellent foundation for UnflodTruth to expand upon!

## Features

- **Real-time Claim Detection**: Monitors and identifies check-worthy factual claims.
- **Instant Evaluation**: Analyzes claims against authoritative sources using AI to provide verdicts.
- **Context Awareness**: Considers the surrounding conversation to improve accuracy.
- **Multilingual Support**: Works across different languages to fact-check globally.

## Getting Started

1. Clone this repository.
2. Follow the setup instructions (coming soon) to install dependencies.
3. Configure your API keys.
4. Run the extension in your browser!
